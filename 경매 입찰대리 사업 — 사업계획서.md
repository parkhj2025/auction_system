# 경매 입찰대리 사업 — 사업계획서

> **서비스 지역**: 인천광역시 (인천지방법원 본원)
> **목표 론칭**: 2026년 5월
> **문서 버전**: v1.3 | 2026년 4월 12일 | 내부 실행용

---

## Executive Summary

경매는 채권자에게는 채무 회수를 위한 유동화 수단이고, 매수자에게는 시가보다 저렴하게 부동산을 취득할 수 있는 합리적 시스템이다. 그러나 "입찰일에 반드시 법원에 직접 가야 한다"는 물리적 제약과, "이 물건을 사도 되는가?"에 대한 정보 비대칭이 참여 장벽으로 작용한다.

본 사업은 이 두 가지 병목을 해소한다. 앞단에서 물건분석·시세비교·수익 시뮬레이션을 무료로 제공하여 경매 접근성을 높이고, 뒷단에서 공인중개사 자격에 기반한 매수신청 대리(입찰 대리) 서비스를 실행한다. 수수료는 5만원부터이며, 퀵서비스처럼 합리적인 가격에 입찰이라는 단일 병목만을 해결한다.

**시장**: 인천지법 연간 경매 접수 10,196건(2024), 개인 입찰 비중 88~91%. 수도권 최저가 시세로 스몰캡 투자자 집중.

**경쟁 우위**: 규제 진입장벽(3종 자격자만 가능) + 콘텐츠 마케팅(바토너 등 기존 서비스에 없는 영역) + 가격 경쟁력(바토너 11만원 대비 55% 저렴).

**성장 경로**: Phase 1 콘텐츠+입찰 대리 검증 → Phase 2 데이터 자산 축적+검증된 파트너 연계 → Phase 3 프롭테크 기업으로 매각.

## L1. Strategy — WHY

### 1.1 Vision & Mission

경매 투자의 정보 비대칭을 해소하여 저변을 확대하고, 물리적 병목(법원 직접 방문)을 합리적 가격에 해결한다.

경매는 채권자와 매수자 양쪽에 합리적인 시스템임에도 불구하고, 권리분석의 난이도, 법적/실무적 불편함, 그리고 법원 직접 방문이라는 시간적 제약이 진입 장벽으로 작용한다. 기존 서비스(바토너 등)는 "입찰이라는 행위의 대행"만을 제공하며, "이 물건을 사야 하는가?"라는 앞단의 판단 재료는 제공하지 않는다.

본 사업은 이 틈새를 공략한다. 앞단에서 물건분석·시세비교·수익 시뮬레이션을 무료 콘텐츠로 제공하여 경매 접근성을 높이고, 뒷단에서 입찰이라는 물리적 행위를 합리적 비용으로 위임할 수 있게 한다. 권리분석·투자자문·명도 등 전문 영역은 서비스 범위에 포함하지 않으며, 전문가 의뢰를 권고한다.

### 1.2 Winning Aspiration

| 단계 | 목표 | 내용 |
|------|------|------|
| Phase 1 | 시장 검증 | 인천 본원 한정. 콘텐츠 파이프라인 + 입찰 대리 단일 서비스. |
| Phase 2 | 데이터 자산 축적 | 시세 변동 추적, 낙찰가율 패턴, 의사결정 도구 구축. 검증된 소수 파트너(법무사 1곳, 인테리어 1곳)와 수직 통합. |
| Phase 3 | 엑시트 | 경매 원스톱 서비스 + 데이터 자산으로 프롭테크/핀테크 기업에 매각. |

### 1.3 Trade-offs (하지 않을 것)

**지역 제한**: 인천지방법원 본원으로 한정한다. 인천에서 수익 구조가 입증된 후에만 확장한다.

**서비스 제한**: 권리분석, 투자자문, 명도를 직접 수행하지 않는다. Phase 1에서는 입찰 대리 단일 서비스만 운영한다.

**시스템 제한**: 처음부터 풀서비스 웹페이지를 구축하지 않는다. MVP 이후에는 양질의 콘텐츠 생산에 집중하고, 수동 운영에서 병목이 터지는 시점에 해당 병목만 자동화한다.

## L2. Market — WHO

### 2.1 Customer Segments

**Segment A: 경매 경험자** — 권리분석이 가능하고 물건 선정도 완료했으나, 입찰일에 법원을 방문할 시간이 없다. 마케팅 접근: 서비스 존재 자체를 알리는 것이 핵심. "5만원에 똑같은 일을 해주는데 시세 비교까지 해준다"가 전환 포인트.

**Segment B: 재테크 관심자** — 경매에 관심은 있으나 장벽을 느끼는 사람. 바토너는 이 사람들을 고객으로 만들 방법이 없다(사건번호 입력이 첫 화면). 우리 블로그가 경매 접근성을 높이고 → 관심 → 이해 → 시도까지 유도하는 퍼널 역할을 한다.

### 2.2 Problem / Jobs to Be Done

| 과업 유형 | 내용 |
|---------|------|
| 기능적 과업 | 내가 원하는 물건에 입찰하되, 법원에 직접 가지 않고 싶다. |
| 경제적 과업 | 기회비용이 대리 수수료보다 클 때, 합리적으로 위임하고 싶다. |
| 감정적 과업 | 중요한 입찰을 신뢰할 수 있는 전문가에게 맡기고 싶다. |
| 정보 과업 | 이 물건이 투자할 가치가 있는지 판단할 재료가 필요하다. |

### 2.3 Existing Alternatives & 포지셔닝 갭

| 대안 | 장점 | 단점 |
|------|------|------|
| 직접 방문 | 비용 없음 | 시간/기회비용 발생 |
| 바토너 (플랫폼) | 전국, 시스템 완비 | 11만원, 콘텐츠 없음, 품질 분산 |
| 런포유 | 8.9만원 | 베타 단계, 변호사 위임 방식 |
| 풀서비스 (법무사) | 원스톱 | 감정가 1~1.5% (고가) |
| 포기 | 시간/비용 절약 | 투자 기회 상실 |

**포지셔닝**: "직접 가기"와 "고가 풀서비스" 사이, 바토너보다 55% 저렴하면서 콘텐츠 가치를 추가한 위치.

### 2.4 Market Size

**TAM**: 인천지법 연간 부동산 경매 접수 10,196건 (2024년). 전년 대비 40.2% 증가.

**SAM**: 전체 낙찰의 88~91%가 개인 입찰자. 약 9,000건/년.

**경쟁 지표**: 물건당 평균 응찰자 6.8명, 매각가율 79.6~80.3% (2026년 1분기, 인천 아파트 기준).

## L3. Value — WHAT

### 3.1 Value Proposition

> "이 물건, 살 만한가요? 숫자로 보여드립니다. 입찰은 5만원이면 됩니다."

앞단(정보 제공): 물건분석·시세비교·수익 시뮬레이션을 무료 콘텐츠로 제공.
뒷단(실행): 입찰이라는 물리적 행위를 합리적 가격에 대리.

바토너는 뒷단만 제공한다. 우리는 둘 다 제공한다.

### 3.2 Solution / Product

공인중개사법에 따른 매수신청 대리(입찰 대리) 서비스. 전자본인서명확인서 기반 위임장으로 비대면 서류 처리가 가능하며, 보증금은 전용계좌로 관리한다.

### 3.3 Moat (경쟁 우위)

**규제 진입장벽**: 변호사, 법무사, 등록 공인중개사만 수행 가능. 등록에는 실무교육 이수, 보증보험 가입, 지방법원장 등록 필요.

**콘텐츠 마케팅 선점**: 바토너를 포함한 기존 서비스에 블로그, 물건분석, 교육 콘텐츠가 전혀 없다. "오늘의 무료 물건분석" 시리즈로 인천 경매 검색 유입을 선점한다.

**가격 경쟁력**: 바토너 11만원 대비 55% 저렴. D-1 패스트트랙 비교: 바토너 15.4만원 vs 우리 10만원.

**데이터 자산**: 크롤러+콘텐츠 파이프라인으로 인천 경매 시장의 시세 데이터·낙찰 패턴·물건분석 아카이브 축적. 시간으로만 쌓이는 Moat.

### 3.4 경쟁사 비교

| 기준 | 우리 | 바토너 | 런포유 |
|------|------|--------|--------|
| 가격 | 5만~10만원 | 11만원 | 8.9만원 |
| 지역 | 인천 전문 | 전국 | 6개 도시 |
| 콘텐츠 | 물건분석 무료 | 없음 | 없음 |
| 시스템 | 수동(Phase 1) | 풀 자동화 | 베타 |
| 고객 접점 | 대표 직접 소통 | 시스템 매칭 | 담당자 배정 |
| Segment B 공략 | ✅ (콘텐츠) | ✗ | ✗ |
| D-1 할증 | 없음 | +4.4만원 | 미확인 |

**바토너 핵심 정보**: (주)케이디씨텍, 2023.07 론칭, 전국 230명 대리인, 보증금 누적 2,400억원(사고율 0%), 최대 10억원 보증. 기술 수준은 "잘 만든 웹앱"이며 따라갈 수 없는 기술 장벽은 아님.

**바토너의 구조적 약점(우리의 기회)**: 중개 플랫폼이므로 대리인 품질 통제 어려움, 콘텐츠 마케팅 부재, 지역 밀착 부재.

## L4. Delivery — HOW

### 4.1 Channels

**킬링 콘텐츠**: "오늘의 무료 물건분석" — 투자자 관점의 수익률 뷰를 제공하는 큐레이션. 경매 대리업무의 저변을 확대하는 서비스 제공의 성격으로 무료 물건분석을 제공한다.

**배포 전략**: 원소스 멀티유즈. AI 자동화를 통해 양산 체계 구축.

| 채널 | 역할 |
|------|------|
| 네이버 블로그 | 검색 유입 (SEO). 물건분석 콘텐츠 → 랜딩으로 유도. **Phase 1 핵심 채널.** |
| 랜딩 페이지 | 전환 도구. 가치+가격+절차+CTA. 클로드 코드로 직접 제작. |
| 카카오톡 채널 | CTA 목적지. 1:1 상담 및 접수. |
| 경매장 현장 | 이미 입찰하러 온 사람에게 명함 배포. |

### 4.2 Customer Relationships

**Phase 1**: 1:1 직접 응대. 대표가 카카오톡으로 직접 상담. 시스템 매칭이 아닌 "얼굴 있는 서비스".

**리텐션**: 포인트 제도 (10회 이용 시 1회 무료 = 10% 할인 + 잠금 효과). Phase 2에서 시스템화.

### 4.3 Key Activities

| # | 활동 | 내용 |
|---|------|------|
| 1 | 콘텐츠 생산/배포 | 매일. 두인옥션 PDF → 크롤러 시세 수집 → AI 콘텐츠 생성 → 블로그 발행. |
| 2 | 상담/전환 | 유입 시 즉시 대응. 카카오톡으로 1:1 응대하여 계약 전환. |
| 3 | 입찰 수행 | 전날 서류 완비. 당일 법원 방문하여 입찰 수행. 결과 즉시 통보. |

**내부 팀**: 박형준님(온라인 마케팅, 사업 세팅, 경영), 정은호님(오프라인 영업, 대면 핸들링), 이기효님(전국 확대 시 인력 관리, 미합류).

**외부 파트너**: 현재 미정. 수요가 증명된 곳에만 파트너를 붙인다. Phase 2에서 인천 특화 법무사 1곳, 인테리어 업체 1곳과 깊은 제휴 — "백화점 식품관"이 아니라 "단골 정육점".

## L6. Growth — WHAT NEXT

### 6.1 Milestones & Phase 전환

Phase 1 → 2 전환 조건: 수요가 공급을 초과하는 상태의 입증. 기준선은 인당 월 수익 1,000만원.

Phase 2 자금 확보: 중소벤처기업부 창업지원 사업에 프롭테크 기업으로 신청.

### 6.2 Growth Engine

**온라인 엔진 (Phase 1 핵심)**: "오늘의 무료 물건분석" 블로그 시리즈. 고객 여정: 블로그 → "이거 넣어볼까" → "여기서 대리입찰도 해주네" → 카카오톡 문의 → 서비스 전환.

**오프라인 엔진**: 경매장 명함 배포, 경매학원 인근 홍보.

### 6.3 Scaling Strategy (Phase 2+)

바토너처럼 중개 플랫폼으로 가지 않는다. 다른 축으로 확장한다.

**축 1 — 콘텐츠 → 데이터 → 의사결정 도구**: 시세 변동 추적(SQLite DB), 낙찰가율 패턴, 사건번호 입력 시 자동 시세 비교+수익 시뮬레이션. 바토너는 사건조회만. 우리는 "이 물건 살 만한가?"까지.

**축 2 — 수직 통합**: 검증된 소수 파트너와 깊은 제휴. 선택지는 적지만 품질 일정.

**축 3 — 지역 독점 후 복제**: 인천 모델 검증 → 수원·대전·부산. 각 지역에 직영 또는 밀착 파트너 1명 + 콘텐츠+데이터 직접 운영.

| | 바토너 | 우리 |
|---|---|---|
| 스케일 방식 | 대리인 수 늘리기 | 지역별 콘텐츠+데이터 복제 |
| 고객 가치 | "편하게 입찰" | "편하게 입찰" + "뭘 살지 판단" |
| Moat | 네트워크 효과 | 데이터 자산 + 지역 콘텐츠 SEO |

### 6.4 Exit Scenario

데이터 자산(시세 DB, 낙찰 패턴, 물건분석 아카이브) + 검증된 운영 노하우 + 고객 기반을 패키지로 구성하여 프롭테크/핀테크 기업에 매각. 잠재 인수자: 토스, 네이버, 직방 등.

## A. SOP — 업무 프로세스

### A.1 접수 프로세스 (Phase 1)

Phase 1에서는 웹 접수 폼 기반. 고객은 사이트에서 접수를 완결하고, 내부 처리(서류 확인, 보증금 매칭, 입찰 수행)는 수동. 시스템 자동화(DB 저장, PG 결제, 실시간 상태 추적)는 병목이 터지는 시점에 진행.

| # | 단계 | 내용 |
|---|------|------|
| 1 | 웹 접수 | 사건번호, 입찰가, 입찰인 정보 입력. 서류(전자본인서명확인서, 신분증) PDF 업로드. |
| 2 | 수수료 안내 | 접수 완료 화면에서 수수료 금액·전용계좌 표시. 계좌이체로 수령. |
| 3 | 보증금 안내 | 접수 완료 화면에서 보증금 금액·전용계좌·마감 시한 표시. **전용계좌**로 D-1 오후까지 입금. |
| 4 | 접수 확정 | 서류+수수료+보증금 확인 → 카카오톡으로 확정 알림 |

### A.2 보증금 처리 모델

**최종 확정 방식**: 고객이 모바일뱅킹으로 전용계좌에 송금 1회. 끝.

**법적 쟁점**: 보증금 수령에 대해 직접적으로 금지하는 명문 규정 없음, 허용하는 명문 규정도 없음. 바토너(업계 최대)는 동일 방식을 공개 운영 중이며 수년간 문제 없음.

**리스크 해소 구조**:
- 보증금 전용계좌 분리 (매출 계좌와 완전 분리)
- 보증보험 가입 (서울보증보험)
- 위임계약서에 전용계좌 명시, 혼용 금지 조항
- 건별 매칭 대장 관리
- 국민신문고 유권해석 확보 (사업 개시 전)

**세무 처리**: 보증금은 예수금(유동부채)으로 처리. 부가세·소득세 대상 아님. 전용계좌 출금이 "법원 납부" 또는 "고객 반환"뿐이면 문제없음.

| 시점 | 차변 | 대변 |
|------|------|------|
| 고객 입금 시 | 보통예금(전용계좌) | 예수금 |
| 낙찰 → 법원 납부 | 예수금 | 보통예금(전용계좌) |
| 패찰 → 고객 반환 | 예수금 | 보통예금(전용계좌) |

| 2 | 당일 법원 방문 | 인천지방법원 경매법정. 보증금으로 자기앞수표 발행 → 봉입. |

### D.5 블로그·홈페이지 하단 고지문 (4항목)

**면책고지**: 본 콘텐츠는 대법원 경매정보 및 두인옥션 공공데이터를 기초로 작성된 참고 자료이며, 투자 권유가 아닙니다. 투자 판단에 대한 책임은 본인에게 있습니다.

**업무범위**: 본 서비스는 공인중개사법에 따른 매수신청 대리(입찰 대리) 업무만을 수행하며, 권리분석·투자자문·명도 등의 업무는 포함되지 않습니다.

**시세정보**: 주변 시세는 네○○ 부동산, 국토부 실거래가 공개시스템 등 외부 데이터를 참조하였으며, 실시간 시세와 차이가 있을 수 있습니다.

**전문가 의뢰 권고**: 본 콘텐츠에 포함된 수익 시뮬레이션은 공개된 시세 데이터를 기초로 산출한 참고 수치이며, 실제 투자 진행 시에는 반드시 법무사·변호사 등 전문가의 권리분석과 자문을 거치시기 바랍니다.

- 바토너 웹사이트 정밀 분석 (웹사이트 트리, 가격 체계, 서비스 플로우, 신뢰 구조, 전문가 서비스)
- 런포유, 풀서비스형, 무료/카페형 비교
- 비교우위 5축 도출: 가격, 콘텐츠, 지역 밀착, 직접 소통, D-1 대응

**L6 확장 전략 변경**
- 기존: Phase 2 = 경락대출/명도/인테리어 연계 원스톱
- 변경: Phase 2 = 데이터 자산 축적 + 검증된 소수 파트너 수직 통합 + 지역 독점 후 복제
- 근거: 바토너를 따라가지 않는 확장 방향. 시스템 자동화보다 데이터 자산이 진짜 Moat.

**SOP 보증금 처리모델 추가 (섹션 A.2)**

- 법적 쟁점 분석: 명문 금지·허용 모두 없음, 바토너 선례
- 리스크 해소 5항목, 예수금 회계 처리
- 근거: 보증금 전달 방식의 법적·세무적·운영적 검토 완료.

**Phase 1 기술·운영 수준 신설 (섹션 C)**
