## 2. 컴플라이언스 수정 (4건)

### C1. 비교급 가치판단 어휘 ⚠️

| 항목 | 내용 |
|------|------|
| 위치 | 법원경매정보 사이트 첫 문단 |
| 수정 전 | 정보의 정확성과 실시간성이 가장 높고 |
| 수정 후 | 법원이 직접 관리하는 1차 자료이며 |
| 이유 | "가장 높고"는 다른 사이트(유료 경매 정보 등)와의 비교급. 1차 자료라는 사실 표현으로 대체. |

### C2. "보증" 어휘 회피 ⚠️

| 항목 | 내용 |
|------|------|
| 위치 | 매각물건명세서 설명 |
| 수정 전 | 법원이 공식적으로 보증하는 권리관계 자료 |
| 수정 후 | 법원이 공식 작성한 권리관계 정리 자료 |
| 이유 | "보증"은 법적으로 책임 소재 표현이 강한 단어. 실제 매각물건명세서에 잘못 기재된 사항이 있을 경우 매각불허가 신청 사유는 되지만 "법원의 보증"은 부정확한 표현. "공식 작성"으로 사실 기술. |

### C3. 서류 비치 시기 정확화 ⚠️

| 항목 | 내용 |
|------|------|
| 위치 | 세 가지 핵심 서류 도입부 |
| 수정 전 | 매각물건명세서 1주 전 / 현황조사서·감정평가서 2주 전 (서로 다른 시기) |
| 수정 후 | 셋 다 매각기일 1주 전부터 (민사집행규칙 §55) |
| 이유 | 일부 자료에서 "현황조사서·감정평가서 2주 전"으로 적힌 경우가 있으나, 민사집행규칙 §55는 셋 모두 "매각기일(기간입찰의 경우 입찰개시일) 1주 전까지 비치"로 통일. 법조문 정확성 우선. |

### C4. 권리분석 경계 회피 ⚠️

| 항목 | 내용 |
|------|------|
| 위치 | 매각공고 항목 설명 |
| 수정 전 | (초안에 "권리분석에서 핵심" 표현 포함됨) |
| 수정 후 | (해당 표현 삭제 → "입찰가를 정할 때 고려해야 하는 항목") |
| 이유 | 헌법 §1 트레이드오프: 권리분석은 우리 서비스 범위 외. "권리분석에서 핵심"이라는 표현은 우리가 권리분석을 하고 있다는 인상을 줌. 사실 표현으로 대체. |

| **CTA 부재** | 의도된 설계. CTA는 사이트 레벨 컴포넌트로 별도 처리. |

## 8. 툴팁 후보 7건

(디자인 라운드에서 활용)

| # | 용어 | 정의 (1~2문장 제안) |
|---|------|-------------------|
| 1 | 사건번호 | 경매 사건을 구분하는 고유 식별자. "연도+사건부호+일련번호" 형식으로 구성된다. |
| 2 | 물건번호 | 한 사건에 여러 부동산이 묶여 있을 때 각 부동산을 구분하는 번호. 같은 사건이라도 물건번호가 다르면 별도 입찰한다. |
| 3 | 매각공고 | 매각기일 약 14일 전, 법원이 매각 정보를 공개적으로 알리는 절차. 민사집행법 §106이 정한 7가지 항목이 포함된다. |
| 4 | 최저매각가격 | 그 회차에서 입찰 가능한 최소 가격. 이보다 낮은 입찰은 무효 처리된다. |
| 5 | 매각물건명세서 | 부동산의 권리관계를 정리한 법원 공식 문서. 매각으로 소멸되지 않는 권리, 점유자 정보 등이 기재된다. 매각기일 1주 전부터 열람 가능. |
| 6 | 현황조사서 | 집행관이 현장에 직접 나가 조사한 보고서. 점유 현황, 임차인의 전입신고 및 확정일자 등이 기록된다. 매각기일 1주 전부터 열람 가능. |
| 7 | 감정평가서 | 감정인이 부동산 가격을 평가한 서류. 평가액과 평가 근거, 외관 정보 등이 포함된다. 매각기일 1주 전부터 열람 가능. |
