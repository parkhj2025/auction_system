## 2. 컴플라이언스 수정 (3건)

### C1. 가치판단 어휘 회피 ⚠️

| 항목 | 내용 |
|------|------|
| 위치 | 인도명령 |
| 수정 전 | 별도의 본안 소송 없이 법원의 결정만으로 진행되어 비교적 빠르게 처리됩니다. |
| 수정 후 | 별도의 본안 소송 없이 법원의 결정만으로 진행됩니다. |
| 이유 | "비교적 빠르게"는 명도소송과의 비교급 가치판단. 사실 표현(별도 소송 없이 결정만으로)으로 메시지 충분 전달. |

### C2. 비교 어휘 정정 ⚠️

| 항목 | 내용 |
|------|------|
| 위치 | 명도소송 |
| 수정 전 | 본안 판결까지 시간이 더 걸립니다 |
| 수정 후 | 본안 판결을 받기까지 별도의 시간이 소요됩니다 |
| 이유 | "더 걸린다"는 인도명령과의 비교 표현. "별도의 시간이 소요"는 사실 기술. |

### C3. 법령 수치 회피 ⚠️ (중요)

| 항목 | 내용 |
|------|------|
| 위치 | 대금 미납의 결과 — 재매각 취소 |
| 수정 전 | 매각대금, 지연이자(연 12%), 절차비용을 모두 지급하면 |
| 수정 후 | 매각대금, 법령에 따른 지연이자, 절차비용을 모두 지급하면 |
| 이유 | 자료마다 지연이자율 표기가 상이함 — 찾기쉬운생활법령정보 자료 일부에 "연 1할 5푼(15%)" / 일부에 "연 12%" / 또 다른 자료에 "연 2할(20%)"로 다르게 명시. 시기별 개정 또는 자료 오류 가능성. **수치 단정 시 사실 오류 위험**이 있어, "법령에 따른 지연이자"로 안전 회피. 정확한 수치는 디자인 라운드 또는 별도 검증 후 확정. |

## 4. 마케터 검수 결과

### 통과 항목

| 항목 | 평가 |
|------|------|
| **헤드라인 SEO** | "낙찰 이후 — 매각허가결정에서 인도까지" — 검색 매칭 자연 |
| **인트로 후킹** | "가장 높은 가격을 적었다고 해서 즉시 내 것이 되는 것은 아닙니다" — 입문자 오해 즉시 교정, 호기심 유발 |
| **시간 순서 흐름** | 매각허가결정 → 확정 → 잔금 → 등기 → 인도 → 대금 미납 — 매수인 시점 시간 순서 |
| **시리즈 마무리 표** | 6편 회차별 핵심 내용 표 → 시리즈 전체 한눈 회상 가능 |
| **입찰 전 점검 2가지** | 매각으로 소멸되지 않는 권리 + 대항력 임차인 — 시리즈 전체 핵심을 2가지로 압축 |
| **Segment B 어필** | "소유권을 취득해도 부동산 안에 점유자가 계속 있으면 실제로 사용할 수 없습니다" — 직관적 설명 |
| **Segment A 어필** | 즉시항고 보증 공탁(매각대금 1/10), 등기 촉탁 첨부서류, 재매각 취소 조건 — 디테일 풍부 |
| **마무리** | "권리분석과 시세 평가는 본 시리즈의 범위를 벗어나므로 전문가 자문을" — 헌법 §1 트레이드오프 + 컴플라이언스 일관 |

수정 사항 없음.

### 시리즈 마무리 효과

6편으로 시리즈를 닫는 두 가지 장치:

1. **회차별 정리표** — 1편에서 6편까지 다룬 핵심을 압축. 처음 본 독자가 어디서 시작할지 알 수 있고, 다 읽은 독자가 자기가 무엇을 봤는지 회상할 수 있음.
2. **입찰 전 점검 2가지** — 시리즈 전체 정보를 두 가지 행동 항목으로 압축. "그래서 결국 뭘 봐야 하나"에 대한 답.

## 8. 툴팁 후보 (6편 신규 5개 + 시리즈 누적)

### 6편 신규

| # | 용어 | 정의 (1~2문장 제안) |
|---|------|-------------------|
| 1 | 매각결정기일 | 법원이 매각을 허가할지 여부를 결정하는 기일. 매각기일로부터 1주 이내에 열린다. |
| 2 | 매각불허가결정 | 매각기일에 정해진 최고가매수신고인에게 매각을 허가하지 않기로 하는 법원의 결정. 매수신고인은 매수신청보증을 돌려받는다. |
| 3 | 즉시항고 | 법원의 결정에 불복해 상급심에 다시 판단을 구하는 절차. 매각허가/불허가결정에 대해서는 결정 선고일로부터 1주 이내에 신청해야 한다. |
| 4 | 대금지급기한 | 매수인이 잔금을 납부해야 하는 기한. 매각허가결정 확정일로부터 1개월 안의 날로 법원이 정한다. |
| 5 | 명도소송 | 부동산 점유자에게 부동산을 비울 것을 명하는 본안 판결을 받기 위한 소송. 인도명령 대상이 아닌 점유자(대항력 있는 임차인 등)에 대해 거치는 절차다. |
